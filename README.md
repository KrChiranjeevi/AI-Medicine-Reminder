Problem Statement:
People often forget to take their medicines on time. This app provides a voice reminder to ensure timely medication.


Brief Description:
Our project is an AI-powered web-based medication reminder that allows users to set medicine reminders with voice alerts.

- Languages: Python, HTML, CSS, JavaScript
- Framework: Flask
- Tools/Platform: VS Code, GitHub
- API: Web Speech API (Text-to-Speech)

- Add name, medicine, and time
- Schedule multiple reminders
- Voice alerts using browser
- Simple web interface

Benefits:
- Helps patients remember medicine timings
- Works on any browser
- Easy to customize
